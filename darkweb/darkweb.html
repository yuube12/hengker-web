ğŸ“Œ Apa Itu OpenVPN?

  • openvpn adalah aplikasi client untuk menjalankan koneksi VPN.
  • Biasanya sudah terinstal di Kali Linux. Cek dengan:
  • openvpn --version
  • Kalau hanya mengetik openvpn, akan muncul manual-nya.

Tapi OpenVPN tidak bisa digunakan langsung, kamu perlu:

  • Konfigurasi file .ovpn dari penyedia VPN (seperti VPNBook)
  • Username dan password (kalau perlu)

ğŸ”§ Langkah-Langkah Menggunakan VPNBook di Kali Linux

  1. Kunjungi situs: vpnbook.com
  2. Klik tab "Free VPN" di atas
  3. Scroll ke bagian "OpenVPN Config Files"
  4. Download beberapa file ZIP, misalnya:
    • vpnbook-us1-tcp80.ovpn.zip
    • vpnbook-ca1-udp25000.ovpn.zip, dll

    Karena kadang satu server penuh

  5. Ekstrak file ZIP-nya
  6. Buka terminal dan masuk ke folder hasil ekstrak:

    7. cd ~/Downloads/vpnbook-.../

  7. Jalankan file .ovpn

    8. sudo openvpn vpnbook-(tulis namanya).ovpn

  8. Masukkan username: vpnbook
  9. Password bisa kamu lihat di halaman yang sama (bagian bawah). Selalu berubah, jadi harus cek ulang.
  10. Jika berhasil, akan muncul:

    11. Initialization Sequence Completed

    Artinya: VPN aktif

  11. Kalau tidak bisa, berarti server penuh. Hentikan openvpn dengan: (di terminal lain)
  12. sudo pkill openvpn

    13. Perintah ini akan mencoba menghentikan semua proses bernama openvpn.

  13. Coba hubungkan lagi hingga berhasil.
  14. Setelah berhasil, coba akses internet, misalnya buka Google. Jika tidak bisa, lakukan perbaikan DNS.

ğŸ”„ï¸ Cara Mengganti DNS di Kali Linux

File DNS default adalah /etc/resolv.conf, namun biasanya dilindungi dengan atribut "immutable" agar tidak bisa diubah seenaknya.

  1. Cek status proteksi file:

    2. sudo lsattr /etc/resolv.conf

    jika ada huruf i, artinya file dikunci

  2. Hapus proyeksi

    3. sudo chattr -i /etc/resolv.conf

  3. Edit file DNS

    4. sudo nano /etc/resolv.conf

  4. Tambahkan dengan DNS yang kamu pilih

    5. nameserver 8.8.8.8 # Google DNS
    nameserver 8.8.4.4 # Google DNS cadangan
    nameserver 1.1.1.1 # Cloudflare DNS (lebih aman dan privat)

    Lalu berikan DNS asli tanpa tanda pagar #

  5. Simpan dengan:
    • Ctrl + O, lalu Enter
    • Ctrl + X, untuk keluar

â— Peringatan:

  • DNS akan kembali ke asli saat reboot (kecuali kamu ubah permanen).
